# 💰 Finance Tracker

A modern and responsive Finance Tracker web application built using HTML, CSS, and JavaScript. This project helps users manage their personal finances by tracking income, expenses, budgets, savings goals, and financial reports through an intuitive dashboard.

## 🚀 Features

### 🔐 User Authentication
- Login Page
- Sign Up Page
- Form Validation
- Secure User Navigation

### 📊 Dashboard
- Financial Overview
- Total Income
- Total Expenses
- Current Balance
- Quick Statistics

### 💵 Income Management
- Add Income Sources
- View Income Records
- Track Earnings

### 💸 Expense Tracking
- Add Expenses
- Categorize Spending
- Monitor Expense History

### 📈 Budget Management
- Create Monthly Budgets
- Track Budget Usage
- Budget Progress Indicators

### 🎯 Financial Goals
- Set Savings Goals
- Track Goal Progress
- Manage Multiple Goals

### 🏦 Accounts
- Manage Multiple Accounts
- Account Balance Tracking
- Financial Summary

### 📋 Transactions
- Complete Transaction History
- Income & Expense Records
- Easy Transaction Management

### 📑 Reports & Analytics
- Financial Reports
- Spending Analysis
- Income vs Expense Overview
- Data Visualization

### ⚙️ Settings
- User Preferences
- Account Settings
- Application Configuration

---

## 🛠️ Technologies Used

- HTML5

- JavaScript (ES6)
- Local Storage
- Responsive Web Design
